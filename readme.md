# 알고리즘 및 C++ 학습 저장소

이 저장소는 C++ 문법 기초부터 다양한 알고리즘 문제를 해결하며 공부한 과정을 기록하는 곳입니다. 단순히 정답을 맞히는 것에 그치지 않고 실무에서도 통용될 수 있는 유지보수가 용이하고 가독성 있는 안정적인 코드를 작성하는 기술을 숙달하는 것을 목표로 합니다.

## 저장소 구조

### 1. 이론 및 기초 학습 (Learn)
C++를 통하여 자료구조, 알고리즘 이론을 주제별로 구현하며 학습한 기록입니다.
* **기본 문법**: C++ 컨벤션, 배열 및 벡터 활용
* **자료구조**: 연결 리스트, 스택, 큐 등 기초 자료구조 직접 구현
* **알고리즘 이론**: 정렬(Quick, Merge), 이진 탐색, 그래프 순회, 동적 계획법(DP) 등

### 2. 문제 풀이 (날짜별 폴더)
꾸준히 풀어온 백준(BOJ), 프로그래머스 등의 문제 풀이 기록입니다.
* 폴더명은 `YYMMDD` 형식으로 관리하였고, 각 파일의 번호는 백준의 문제 번호입니다.
* 한번 푼 문제도 애매할 경우 다시 풀어서 복습을 하며 이해하려 노력했습니다.

### 3. 언어 확장 (Rust)
C++ 외에도 Rust를 통해 문제를 해결해보려고 하였습니다.

## 사용 환경

* **주력 언어**: C++ (C++17 표준)
* **도구**: Clang / VS Code
* **주요 학습처**: 백준(BOJ), 프로그래머스
